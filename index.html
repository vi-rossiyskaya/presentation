Alternatives to JSON: YAML/TOML/... and why

JSON

JSON stands for JavaScript Object Notation

JSON is a lightweight format for storing and transporting data

JSON is often used when data is sent from a server to a web page

JSON is "self-describing" and easy to understand

JSON Syntax Rules

Data is in name/value pairs

Data is separated by commas

Curly braces hold objects

Square brackets hold arrays


              {
                "books": [
                  {
                    "id": "bk102",
                    "author": "Crockford, Douglas",
                    "title": "JavaScript: The Good Parts",
                    "genre": "Computer",
                    "price": 29.99,
                    "publish_date": "2008-05-01",
                    "description": "Unearthing the Excellence in JavaScript"
                  }
                ]
              }

YAML

YAML Ain't Markup Language

YAML is a human friendly data serialization standard for all programming languages.

YAML Syntax Rules


              ---
              books:
              - id: 'bk101'
                author: 'Crockford, Douglas'
                title: 'JavaScript: The Good Parts'
                genre: 'Computer'
                price: 29.99
                publish_date: '2008-05-01T00:00:00+00:00'
                description: 'Unearthing the Excellence in JavaScript'
              ...

JSON advantages compare to YAML

Easy to learn and adopt

Fast and easy to validate

Compact

Secure

YAML advantages compare to JSON

Handles a variety of tasks

Works with multiple data types

Allows comments

Uses natural language

TOML

a minimal configuration file format

is designed to map unambiguously to a hash table

easy to parse into data structures

TOML has useful native types

Key/Value Pairs

Arrays

Tables

Inline tables

Arrays of tables

Integers & Floats

Booleans

Dates & Times, with optional offsets


              [[books]]
              id = 'bk101'
              author = 'Crockford, Douglas'
              title = 'JavaScript: The Good Parts'
              genre = 'Computer'
              price = 29.99
              publish_date = 2008-05-01T00:00:00+00:00
              description = 'Unearthing the Excellence in JavaScript'

Arrays in TOML


              int_values = [1, 2, 4, 8, 16]
              strings = ["prime", "audio", "soup"]
              mixed = [1, 2, "Three", 4.0]
              multi-line = [
                  "array",

              ]
